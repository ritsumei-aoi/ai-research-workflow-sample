# Experiments

This directory contains experimental scripts and quick analyses used during development.

Current contents:

- scripts to compute determinant distributions are in scripts/ (compute_and_save_det.py, compute_n4_sample.py, compute_n5_sample.py)

Planned additions:

- experiments/compute_summary.py: utilities to load JSON results and produce summary tables and plots.
- experiments/exhaustive_n4_plan.md: resource estimate and plan for exhaustive n=4 computation.
