# Experiments

This directory contains experimental scripts and quick analyses used during development.

## Current contents

- compute_and_save_det.py: Compute and save determinant distributions for n=1,2,3 (exhaustive) and n=4,5 (sampling)
- compute_n4_sample.py: Sample-based computation for n=4
- compute_n5_sample.py: Sample-based computation for n=5

## Planned additions

- compute_summary.py: Utilities to load JSON results and produce summary tables and plots
- exhaustive_n4_plan.md: Resource estimate and plan for exhaustive n=4 computation

All scripts in this directory are for demonstration, validation, or exploratory analysis. See each script's docstring for usage.
