# Known Results on (-1,0,1)-Matrices

本ドキュメントでは，三値行列（要素が -1, 0, 1）の既知の性質・最大行列式・個数等について，主要な外部情報源をまとめます。

## 参考文献・リンク
- MathWorld: https://mathworld.wolfram.com/-101-Matrix.html
- OEIS A051753: https://oeis.org/A051753

## 主な性質
- n×n の (-1,0,1)-行列の最大行列式は Hadamard 行列と同じく 1, 2, 4, 16, 48, ...（OEIS A003433）
- 最大行列式を持つ n×n (-1,0,1)-行列の個数: 1, 4, 240, 384, 30720, ...（OEIS A051753）
- 行・列和が 2n 種類となる場合の個数や性質も研究されている（詳細は MathWorld 参照）
