# 行列式の基礎理論

本ドキュメントでは、行列式の定義と基本的な性質についてまとめます。

## 1. 行列式の定義

$n 	imes n$ 行列 $A = (a_{ij})$ の行列式 $\det(A)$ は、以下のライプニッツの公式によって定義されます。

$$ \det(A) = \sum_{\sigma \in S_n} \left( \mathrm{sgn}(\sigma) \prod_{i=1}^n a_{i, \sigma(i)} 
ight) $$

ここで、$S_n$ は $\{1, 2, \dots, n\}$ のすべての順列の集合、$\mathrm{sgn}(\sigma)$ は順列 $\sigma$ の符号（偶順列なら +1、奇順列なら -1）を表します。

小規模な行列の場合、以下のように計算できます。

- $2 	imes 2$ 行列 $A = \begin{pmatrix} a & b \ c & d \end{pmatrix}$ の場合：
  $ \det(A) = ad - bc $

- $3 	imes 3$ 行列 $A = \begin{pmatrix} a & b & c \ d & e & f \ g & h & i \end{pmatrix}$ の場合：
  $ \det(A) = a(ei - fh) - b(di - fg) + c(dh - eg) $ (サラスの規則も参照)

## 2. 行列式の基本的な性質

1.  **行/列の線形性:** 行列式は、各行（または各列）に関して線形性を持つ。
    -   ある行を $c$ 倍すると、行列式も $c$ 倍になる。
    -   ある行が2つのベクトルの和である場合、行列式はそれぞれのベクトルを行として持つ行列式の和になる。

2.  **行/列の交換:** 2つの行（または列）を入れ替えると、行列式の符号が反転する。

3.  **同じ行/列:** 2つの行（または列）が同じである場合、行列式は 0 となる。

4.  **行/列の操作:** ある行（または列）の定数倍を別の行（または列）に加えても、行列式の値は変わらない。

5.  **転置:** 行列 $A$ の転置行列 $A^T$ の行列式は、$A$ の行列式に等しい。
    $ \det(A^T) = \det(A) $

6.  **積の規則:** 2つの $n 	imes n$ 行列 $A, B$ について、積の行列式はそれぞれの行列式の積に等しい。
    $ \det(AB) = \det(A)\det(B) $

7.  **逆行列:** 行列 $A$ が正則（逆行列を持つ）である場合、その行列式は 0 ではない。逆行列 $A^{-1}$ の行列式は $1/\det(A)$ に等しい。
    $ \det(A^{-1}) = 1/\det(A) $

8.  **対角行列・三角行列:** 対角行列や三角行列の行列式は、対角成分の積に等しい。

## 3. 余因子展開

行列式は、任意の行 $i$ または列 $j$ に関する余因子展開によっても計算できます。

-   **行 $i$ に関する展開:**
    $ \det(A) = \sum_{j=1}^n a_{ij} C_{ij} $
-   **列 $j$ に関する展開:**
    $ \det(A) = \sum_{i=1}^n a_{ij} C_{ij} $

ここで、$C_{ij} = (-1)^{i+j} M_{ij}$ は余因子であり、$M_{ij}$ は $A$ から $i$ 行目と $j$ 列目を除いた小行列の行列式（小行列式）です。
