## Implementation Guidelines (実装ガイドライン)

- 行列式は整数として厳密計算（浮動小数点は使用しない）
- n ≤ 4 では全数探索、n ≥ 5 ではランダムサンプリングを併用
- JSON スキーマにはバージョン番号 `schema_version` を含める
- 結果の再現性のため、乱数シードを記録する
